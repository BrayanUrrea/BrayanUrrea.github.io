¿Qué es?

Es una applet para crear páginas web desde cero sin necesidad de saber algo de programación, esta plataforma le permitirá al usuario arrastrar los elementos que necesite y estará a su disposición para la creación de su página.

¿Que utilizamos de lo visto en clase?

Usamos varios tipos de las cosas vistos en clases como añadir una imagen, crear tablas , añadir hipervinculos el manejo de espacio en una pagina web y agregar slider para ver el paso de varias imagenes.

¿Cual es el proposito?

Con esta plataforma la idea es guiar a los usuarios que esten en proceso de aprender a diseñar paginas web con unos instrumentos que nosotros les brindamos para aprender con nuestros tutoriales.
