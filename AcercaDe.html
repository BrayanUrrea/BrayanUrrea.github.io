¿Qué es?

Es una applet para crear páginas web desde cero sin necesidad de saber algo de programación, esta plataforma le permitirá al usuario arrastrar los elementos que necesite y estará a su disposición para la creación de su página.

¿Cómo se utiliza?

Esta applet es una herramienta muy intuitiva que le brinda la posibilidad de crear páginas estructuralmente y modificar los estilos a su gusto; poniendo el tamaño a los elementos para una mejor apariencia del sitio y reajustar los elementos para una mejor visualización.

¿A quién va dirigido?

Esta plataforma está dirigida para las personas que saben algo de diseño de paginas web o que están en el proceso de aprendizaje; con el fin de usos institucionales.
