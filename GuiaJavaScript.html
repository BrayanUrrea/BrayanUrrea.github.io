Comando Descripcion Ejemplo
assert(expression, message) Envía un mensaje si se expressionevalúa como false . console.assert((x == 1), "assert message: x != 1");
clear() Borra los mensajes de la ventana de la consola, incluidos los mensajes de error de secuencia de comandos, y también borra el script console.clear();
count(title) Envía el número de veces que el comando count fue llamado a la ventana de consola. console.count(); console.count("inner loop");
debug(message) Se ingresa todo el codigo que se visualiza en una pagina Web <body>...Codigo Visible...</body>
Iniciar Envía messagea la ventana de consola.Este comando es idéntico a console.log. console.debug("logging message");
dir(object) Envía el objeto especificado a la ventana de consola y lo muestra en un visualizador de objetos. console.dir(obj);
dirxml(object) Envía el nodo XML especificado objecta la ventana de consola y lo muestra como un árbol de nodos XML. console.dirxaml(xmlNode);
error(message) nvía messagea la ventana de consola. El texto del mensaje es rojo y precedido por un símbolo de error. console.error("error message");
group(title) Inicia un agrupamiento para los mensajes que se envían a la ventana de la consola y envía el opcional titlecomo una etiqueta de grupo <br>console.group("Level 2 Header"); console.log("Level 2"); console.group(); </br>
groupEnd() Finaliza el grupo actual. Requisitos: Visual Studio 2013 Consulte el ejemplo del groupcomando.
log(message) Envía messagea la ventana de consola. console.log("logging message")
profile(reportName) Se utiliza en aplicaciones web. No se admite en Aplicaciones de almacenamiento mediante JavaScript. No soportado.
Select(element) Selecciona el HTML especificado elementen el Explorador de DOM console.select (elemento)
