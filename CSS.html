CSS

CSS corresponde a la expresión Cascading StyleSheets, que puede traducirse como “Hojas de estilo en cascada”. El concepto se refire a un lenguaje empleado en el diseño gráfico.

El lenguaje CSS permite presentar, de manera estructurada, un documento que fue escrito en un lenguaje de marcado. Se usa especialmente en el diseño visual de un sitio web cuando las páginas se hallan escritas en XML o HTML.

1. Animaciones en CSS :

Comandos Funcionalidad Modo de Uso
Animation-delay Define cuánto tiempo debe esperar
la animación antes de comenzar.
 animation-delay: 1.2s;
Animation-direction Define en qué dirección se
reproduce la animación.
 animation-direction: normal;
Animation-duration Define cuánto dura la animación. animation-duration: 1.2s;
Animation-fill-mode Define lo que sucede antes de que una animación
comience y después de que termine.
 animation-fill-mode: none;
Animation-iteration-count Define cuántas veces se reproduce la animación. animation-iteration-count: 2;
Animation-name Define qué fotogramas clave de animación utilizar. animation-name: none;
Animation-play-state Define si una animación se reproduce o no. animation-play-state: running;
Animation-timing-function Define cómo se calculan los valores entre el
inicio y el final de la animación.
 animation-timing-function: ease;

2. Antecedentes en CSS :

Comandos Funcionalidad Modo de Uso
Background-attachment Define cómo se comportará la imagen de
fondo al desplazarse por la página.
 background-attachment:fixed;
Background-clip Define hasta dónde debe extenderse el
fondo dentro del elemento.
 background-clip:padding-box;
Background-color Define el color del fondo del elemento. background-color: red;
Background-image Define una imagen como fondo del elemento. background-image:
url(/images/jt.png);
Background-origin Define el origen de la imagen de fondo. background-origin:border-box;
Background-position Define la posición de la imagen de fondo. background-position:center center;
Background-repeat Define cómo se repite la imagen de fondo a
través del fondo del elemento.
 background-repeat:repeat;
Background-size Define el tamaño de la imagen de fondo. background-size:120px 80px;
